# Luminang

An educational adventure game exploring Philippine languages and culture through interactive gameplay. Built with Next.js, React, and Tailwind CSS.

## About

Luminang is an immersive game that takes players on a journey through the Philippine archipelago. Explore ancient ruins, decode regional dialects, and unlock the secrets of a lost civilization while learning about Filipino culture and languages.

##  Getting Started

### Prerequisites

- Node.js 18+ installed
- npm, yarn, pnpm, or bun package manager
